\subsubsection{}
$a_0 = 10$,$\displaystyle a_{n+1} = a_n - \frac{1}{a_n}$$(n=0,1,2,\dots)$とする、$a_n$がはじめて負になる$n$を答えよ。\par
答え. 52
\begin{lstlisting}
from fractions import Fraction
def an(n):
    a0 = 10
    while n > 0:
        a0 = a0 - 1 / a0
        n-=1
    return a0

if __name__ == '__main__':
    for i in range(100):
        if an(i) <= 0:
            print('{0} = {1}'.format(i,an(i)))
            break
\end{lstlisting}
